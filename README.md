■ Desafio Técnico -- Desenvolvedor(a) Fullstack Júnior

## Objetivo

Avaliar sua capacidade de programar, estruturar soluções simples, pensar em arquitetura de

software e explicar seu raciocínio.

## ■■■ Testes de Código

### 1. HTML + CSS

Crie uma página simples com:

- Um formulário de cadastro com os campos Nome, E-mail e Senha.

- Um botão de enviar estilizado.

- Regras básicas de validação (por exemplo: campo obrigatório).

■ Explique como organizou seu HTML e CSS.

### 2. JavaScript / Vue.js

Implemente uma pequena lista de tarefas (to-do list) que permita:

- Adicionar novas tarefas.

- Marcar uma tarefa como concluída.

- Remover tarefas da lista.

■ Explique como organizou o código, e se usou Vue.js, descreva os componentes criados.

### 3. TypeScript

Dado o seguinte array de objetos em TypeScript:

const users = [ { id: 1, name: "Ana", age: 25 }, { id: 2, name: "Pedro", age: 30 }, { id: 3, name:

"Maria", age: 22 } ];

Crie uma função que:

- Receba esse array.

- Retorne apenas os nomes dos usuários com mais de 23 anos.

■ Explique como você declarou os tipos e o porquê.

### 4. Banco de Dados (MySQL / SQL)

Considere uma tabela chamada users:

| id | name | email | created_at | |----|-------|--------------------|-------------| | 1 | Ana | ana@email.com |

2024-01-01 | | 2 | Pedro | pedro@email.com | 2024-02-01 | | 3 | Maria | maria@email.com |

2024-02-15 |

Escreva uma query SQL para:

- Listar todos os usuários ordenados pela data de criação (do mais recente para o mais antigo).

■ Explique o funcionamento da sua query.

### 5. Backend (NestJS ou Node.js)

Crie uma rota de API que retorne a lista de usuários em formato JSON.

Endpoint: GET /users Retorno esperado: [ { "id": 1, "name": "Ana", "email": "ana@email.com" }, {

"id": 2, "name": "Pedro", "email": "pedro@email.com" } ]

■ Explique como organizou os arquivos e a estrutura do backend.

## ■■ Testes de Arquitetura e Resolução de Problemas

### 6. Organização de Projeto

Imagine que você vai iniciar um projeto fullstack do zero.

- Quais pastas e arquivos você criaria para organizar o frontend (Vue.js)?

- Quais pastas e arquivos você criaria para organizar o backend (NestJS)?

■ Descreva como faria essa organização e o motivo.

### 7. Cenário de Problema

Você está desenvolvendo uma aplicação e percebe que a tela de login está demorando muito para

carregar.

- Quais seriam as primeiras verificações que você faria para identificar o problema?

- Cite duas possíveis soluções para melhorar a performance.
