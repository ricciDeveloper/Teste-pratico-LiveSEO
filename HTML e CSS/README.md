
Desafio técnico para a vaga de Desenvolvedor(a) Fullstack Júnior

A ideia é conhecer como você programa, organiza soluções e resolve problemas no dia a dia.

Você vai passar por etapas rápidas:

1 - Criar um formulário simples (HTML + CSS).

2 - Fazer uma to-do list (JavaScript/Vue.js).

3 - Escrever uma função em TypeScript para filtrar usuários.

4 - Criar uma query SQL para listar usuários por data.

5 - Montar uma rota de API no backend (Node.js/NestJS).

6 - Explicar como organizaria um projeto do zero.

7 - Sugerir soluções para um problema de lentidão no login.Como organizei meu HTML e CSS?
------------------------------

-   Comecei estruturando o HTML com a tag `<head>` e utilizei estrutura semântica no `<body>` (`<header>`, `<main>`, `<footer>`).\
    Criei a estrutura do formulário com `<form>` e seus campos com tipos definidos (`name`, `email`, `password`).\
    Apliquei regras de validação como: preenchimento obrigatório, e-mail contendo "`@`" e domínio "`...com`", e senha obrigatória.

-   Com o HTML construído e os campos validados, iniciei a estilização em um arquivo CSS externo, utilizando as próprias tags como identificadores para estilização.\
    A estilização foi básica, utilizando as cores da live.
